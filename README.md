# POS System – Full Stack Portfolio Project

A complete Point of Sale (POS) and inventory management system built as a full stack portfolio project.

The system allows businesses to manage products, control inventory, register customers, and process sales through a POS interface.

## Tech Stack

### Frontend
- Angular
- Angular Material

### Backend
- Java
- Spring Boot

### Database
- PostgreSQL

### Infrastructure
- Docker

## Features

- Product management
- Inventory control
- Customer registration
- Point of Sale (POS) system
- Sales reports

## Project Purpose

This project was developed as a portfolio application to demonstrate full stack development skills, including:

- REST API development with Spring Boot
- Frontend application development with Angular
- Database design with PostgreSQL
- Containerization using Docker
- Integration between frontend and backend systems
